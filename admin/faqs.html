Login/Error Codes

Frequently Asked Questions

How Do I Login?

To login - type in the Username and Password supplied by the administrator.
Click the submit button, and provided you have entered the correct details, you will now be able to access the Global Tours Operators system to view Bookings and your Payroll.

Who Can Login?

In order to login, you must already have credentials supplied by the systems administrator.
At the moment, this website does not allow for the creation of new accounts, as our information is sensitive and we must monitor the users accessing this information.

By default, all new users are registered as tour guides, which provides limited access in this portal.
Administrators are granted full access.

What is 'ERROR 001: INVALID CREDENTIALS'?

This error relates to credentials entered being invalid. Either your username or password is incorrect.
Please check again and try again.

What is 'ERROR 002: YOU DO NOT HAVE PERMISSION TO ACCESS THIS PAGE'?

This error occurs when a user has not logged in and attempts to enter the Global Tour Operators portal.
Valid user credentials are required.

What is 'ERROR 003: YOU DO NOT HAVE PERMISSION TO ACCESS THIS PAGE'?

This error occurs when a user is not granted adminsitrator access within the Global Tour Operators portal.
This error would only occur when a Tour Guide user is logged in, as they would require Administrator privileges.

Tours

Administrator Access Required.

How Do I Add a New Tour?

To add a new tour, visit the Tours page, by either selecting Tours on the Homepage, or by selecting Tours on the navigation on the top of the site.
Once you have loaded the page, scroll down the page to the bottom, and you will see a box with a link that will redirect you to add a tour.
To add a tour you must supply a tour name, a price and click the "add tour" button.
Once this has been provided, the page will automatically refresh and you will be able to see the new tour alongside all the tours stored in the database.

Can I delete a Tour?

Yes! A tour can be deleted by simply selecting a tour you would like to delete on the table that allows you to view all of the tours.
You will be asked to confirm if you would like to delete the booking once you have selected the bin icon.
If you want to delete the booking, click "ok" and the table will refresh and the booking that was selected will be deleted.

Is it possible to edit an existing tour?

Yes, it is possible! Just click the pencil button on the table and it will redirect you to an edit section.
You will be able to edit the tour name and price here, and click the submit button provided in order to save your changes.

How Do I Change Active Status for a Tour?

To change the active status of a tour, just click the either the tick or 'x' button to change it.

Set Up - System, Surcharges and Users

Administrator Access Required.

Am I able to change the Tax Rate and Flat Rate of the Payroll System?

Yes, the tax rate and flat rate of the system can very simply be amended by accessing the "setup" page which can be access through the homepage or the navigation bar.
Click System Set Up and the form will load with the current Tax and Flat Rate values which will allow you to make amendments.
Once you have edited the rates, click the update button and the page will refresh with the new values.

How do I create a new surcharge?

To create a new surcharge, access the Surcharge page through the homepage or the navigation bar.
At the bottom of the page there will be a box that will inform the user to create a new surchage. Click this button.
You will then be redirected to insert the new surcharge, by selecting a month and a percentage for the surcharge.

Can I view all of the users that are able to login to this system?

Yes, access the Users page through the homepage or the navigation bar.
A list of the users created will be displayed here.

Bookings

Tour Bookings Created By Users

Can I assign myself (tour guide) or a tour guide (admin) to a Booking?

Both the tour guide and the administrator have the ability to assign a tour guide to a booking.

How do I set a Booking as Completed?

Simply click the 'x' button if visible in order to change a booking to completed.

After a tour has been completed, can I change it back to uncomplete?

No, this is unable to be changed. Please contact your systems administrator if this is an issue.

Payroll

View Your Payroll/ View Tour Guide's Payroll

I am an administrator, how do I view a specific tour guide's payroll?

To view a tour guide's payroll, you will need to select a date range, and then select a tour guide from the list of available tour guides and click submit.
You will then be presented with this tour guide's payroll for the dates selected, if applicable.

As a tour guide, how do I view my payroll?

To view a your payroll, you will need to select a date range, and then click submit.
You will then be presented with your payroll for the dates selected, if applicable.

How do I select a date range?

Simply select a date from, and a date to using the two date input fields presented on the screen and click submit.
